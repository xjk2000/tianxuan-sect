### 命令行直接连接

```bash

# MongoDB
mongosh "mongodb://[USER]:[PASSWORD]@[HOST]:[PORT]/[DATABASE]"

# Redis
redis-cli -h [HOST] -p [PORT] -a [PASSWORD]
```
